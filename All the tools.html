PHOTOSHOP CC

In our project, we used the 2017 release of Adobe Photoshop cc to guarantee the maximum quality of our site and icons and for that we equipped the team with
a 2.12 GHz processor overclockable to 2.66 GHz and it went
good, the software ran pretty smoothly and it was
good and we truly altered all of the screenshot and images.

VISUAL STUDIO 

In terms of coding, we have taken the choice of using Microsoft's Visual Studio, the 2015 release to be exact and it
was our choice because of rather organized interface, built in Api
for reviewing methods and it's ultra useful quick
refactoring settings wich automatically help you with errors,stuff...

BootStrap 4

Bootstrap 4 is one of the most popular HTML,CSS and JS framework editors in the world for building responsive mobile compatible projects,we used it because of the easy interface and huge  scripting database and guides for it that really help with the work and allow us to do such professional looking sites like this one in front of us.
